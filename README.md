# Custom UI — Neon Speedometer

Android ilovasi: **Jetpack Compose** va **`Canvas`** yordamida chizilgan neon uslubidagi **custom spidometr** (0–220 km/h). Tayyor gauge komponenti o‘rniga geometriya va qatlamlar qo‘lda boshqariladi.

## ✨ Xususiyatlar

- Neon **cyan** halo va ichki halqa, **qizil** faol zona (tezlik bilan rang intensivligi o‘zgaradi)
- **Doimiy** qizil igna (`NeedleRed`)
- Shkala: katta chiziqlar (har 20 km/h), kichik chiziqlar (har 10 km/h oralig‘i)
- **Slider** orqali tezlikni boshqarish
- **About** ekrani — loyiha haqida qisqa tushuntirish
